Ejemplo de la etiqueta <a>

Antes de comenzar debemos recordar que son las rutas relativas y las rutas absolutas:

Rutas absolutas

: son aquellas que nos van a dirigir directamente a una pagina web, denominadas URL. En ellas se expresan la dirección completa de una página web

Rutas relativas

: son las rutas que expecifican la ubicación de un archivo o de un recurso relacionado con el archivo HTML actual, así evitamos usar la ruta completa.
